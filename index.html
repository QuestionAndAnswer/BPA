BPA - The Ball Pivoting Algorithm PCL

This an implementation of the Ball Pivoting Algorithm for surface reconstruction (Fausto Bernardini et al. 1999) which uses PCL to handle the 3D cloud information.

The last stable implementation can be found in the releases page (https://github.com/rodschulz/BPA/releases). The outputs of the algorithm are generated in format OFF, which can be opened using several 3D data visualization software (GeomView for example).

More information about the algorithm can be found in the following links.

  • http://www.research.ibm.com/vistechnology/pdf/bpa_tvcg.pdf
  • http://dl.acm.org/citation.cfm?id=614442
